NOTICE

  • 01
    주문내역 및 배송정보 조회 방법

    주문내역 조회
    ① 홈페이지 메인화면 오른쪽 상단 주문내역 클릭
    ② 회원 ID와 비밀번호 입력
    비회원 : 주문시 문자 ⁄ 메일로 발송된 주문번호와 주문서 작성시 입력한 비밀번호 입력
    ③ 주문내역 확인 가능

    배송정보 조회
    위의 ①, ②단계와 동일하며
    주문내역 하단에 배송정보 → 배송조회 클릭

  • 02 상품 주문방법 및 주의사항

    상품 주문방법 및 주의사항
    ① 상품검색
    ② 바로구매 클릭 또는 장바구니에 담기
    ③ 회원 ID 로그인 또는 비회원 주문
    ④ 주문서 작성
    · 휴대폰 본인확인을 통한 성인인증 필수
    · 비회원 : 주문서에 입력한 비밀번호를 기억해 주세요 (주문내역 조회시 필요)
    · 희망수령일 : 수령하고자 하는 일자를 입력 (일요일, 월요일, 휴일은 수령이 불가하오니 주문시 참고)
    ⑤ 결제방법 선택 및 결제 (무통장입금, 신용카드)
    · 무통장 입금자의 경우 현금영수증, 세금계산서 필요시 전하실 말씀에 메모를 남겨 주세요
    · 입금자명 : 실제 입금자 성명으로 입력
    ⑥ 주문 성공 화면 (주문번호)
    ⑦ 비회원 주문인 경우,
    6단계에서 주문번호와 승인번호(카드 결제시)를 꼭 메모해 주세요
    단, 회원인 경우 자동 저장되므로 따로 관리하실 필요가 없습니다

  • 03 배송안내

    평일 주문 : 당일 오후 5시까지 주문 및 입금 완료건 당일배송
    주말, 휴일 주문 : 주말, 휴일 종료 후 발송
    희망 수령일 : 주문서 작성시 수령하고자 하는 일자를 입력
    · 일요일, 월요일, 휴일은 수령이 불가하오니 주문시 참고
    배송비 : 기본 4,000원 ⁄ 60,000원 이상 무료배송 (단, 배송지가 한 곳일 경우)
    · 도서간간 지역은 구매가격과 상관 없이 추가배송비 발생
    배송기간 : 평균, 발송 후 익일 수령가능 ⁄ 도서산간지역은 2~3일 소요
    · 배송업체의 사정이나 기상 악화시 배송이 다소 지연 될 수 있음
